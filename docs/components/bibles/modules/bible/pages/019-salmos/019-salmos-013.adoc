[[v19-13-0]]
=== Salmos, 13
// acf ?, ACF - Almeida Corrigida Fiel

[[v19-13-1]]

^<<v19-13-1,1>>^ Até quando te esquecerás de mim, SENHOR? Para sempre? Até quando esconderás de mim o teu rosto?

[[v19-13-2]]

^<<v19-13-2,2>>^ Até quando consultarei com a minha alma, tendo tristeza no meu coração cada dia? Até quando se exaltará sobre mim o meu inimigo?

[[v19-13-3]]

^<<v19-13-3,3>>^ Atende-me, ouve-me, ó Senhor meu Deus; ilumina os meus olhos para que eu não adormeça na morte;

[[v19-13-4]]

^<<v19-13-4,4>>^ Para que o meu inimigo não diga: Prevaleci contra ele; e os meus adversários não se alegrem, vindo eu a vacilar.

[[v19-13-5]]

^<<v19-13-5,5>>^ Mas eu confio na tua benignidade; na tua salvação se alegrará o meu coração.

[[v19-13-6]]

^<<v19-13-6,6>>^ Cantarei ao Senhor, porquanto me tem feito muito bem.
